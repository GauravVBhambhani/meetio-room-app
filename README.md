## Meeting Room Manager v0.2

**Technologies**:

- `React v16`, `Redux v3.7`
- `Auth0`
- `Cordova v7`
- `Gulp v3.9`
- `eslint v4.10`
- `Jest v20`

## Development setup

- `$ npm install gulp-cli -g`
- create `.env` file based on `.env.sample`
- `$ npm install`
- `$ cd main/ && npm install`
- `$ cd main/ && npm run build`
